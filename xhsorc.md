# 小红书笔记 OCR + 书稿辅助 AI 改写系统

## MVP 产品需求说明书（PRD）

版本：V1.0\
阶段：MVP\
更新时间：2026-03

------------------------------------------------------------------------

# 一、项目背景

在小红书内容生产场景中，大量笔记以 **"文字图片"形式发布**。

用户收集这些笔记后，希望通过 AI 实现以下自动化流程：

1.  将图片中的文字 **OCR识别**
2.  按顺序拼接为完整笔记内容
3.  按固定规则 **大幅度改写**
4.  在文章结尾 **自然融入书籍内容**
5.  自动生成：
    -   改写笔记
    -   小红书推荐正文
    -   小红书标签

并支持 **批量处理多个笔记文件夹**。

------------------------------------------------------------------------

# 二、项目目标

构建一个 Web 系统，实现从 **图片笔记 → OCR → AI改写 → 成品输出**
的完整自动化流程。

MVP 阶段目标：

-   支持图片文件夹 OCR
-   支持书稿书库管理
-   支持 AI 改写
-   支持批量任务
-   支持任务状态管理
-   支持查看 OCR 与 AI 结果

# 五、核心概念

## 笔记文件夹

一个文件夹代表一篇笔记。

特点：

-   文件夹内包含多张图片
-   图片内容为文字
-   图片顺序通过文件名控制

示例：

笔记A/ 图1.jpg 图2.jpg 图3.jpg

排序规则必须采用 **自然排序**：

正确：

图1\
图2\
图3\
图10

错误：

图1\
图10\
图2

------------------------------------------------------------------------

## 书稿

系统允许上传 Word 书稿。

用途：

在改写文章 **最后3--4段** 中自然引入书稿经验内容。

特点：

-   一篇笔记绑定一本书
-   书稿先进入 **系统书库**
-   任务创建时选择对应书稿

------------------------------------------------------------------------

# 六、核心业务流程

## 单任务流程

1.  上传图片文件夹
2.  按文件名排序
3.  逐图 OCR
4.  拼接原创笔记
5.  解析书稿
6.  匹配书稿片段
7.  组装 Prompt
8.  调用 AI 改写
9.  生成最终结果

------------------------------------------------------------------------

## 批量任务流程

1.  创建批次任务
2.  选择多个文件夹
3.  为每个文件夹绑定书稿
4.  生成多个子任务
5.  逐个执行 OCR 与 AI 改写

失败策略：

-   单任务失败不影响其它任务
-   系统记录失败原因

------------------------------------------------------------------------

# 七、OCR 技术方案

MVP 使用：

**PaddleOCR**

原因：

-   中文识别能力强
-   开源
-   支持本地部署
-   文档图片识别成熟
-   支持批量处理

备选方案：

RapidOCR（更轻量）\
Tesseract（通用 OCR）

OCR 规则：

-   按图片顺序执行 OCR
-   每张图独立识别
-   不做自动纠错
-   不做文本清洗
-   保留原识别结果

------------------------------------------------------------------------

# 八、书稿处理

支持格式：

docx

书稿解析流程：

1.  读取 Word 文件
2.  提取正文
3.  自动切片

切片规则：

优先按标题切分\
否则按自然段聚合

每个片段约 **300--800 字**

------------------------------------------------------------------------

# 九、书稿匹配

MVP 阶段使用 **关键词匹配**：

流程：

1.  从原创笔记提取关键词
2.  对书稿片段评分
3.  选择 Top 2--3 段

后续版本可升级：

-   向量检索
-   RAG
-   Embedding

------------------------------------------------------------------------

# 十、AI 改写

输入包含：

-   固定提示词模板
-   原创笔记内容
-   书稿片段
-   书名
-   输出格式要求

改写规则：

保持：

-   段落数量
-   段落顺序
-   小标题
-   列表结构

允许：

-   改变句式
-   改变表达方式
-   调整语气

禁止：

-   改变逻辑
-   改变结构

书籍内容规则：

-   仅在最后 3--4 段出现
-   自然引出
-   不广告化
-   不出现购买引导

------------------------------------------------------------------------

# 十一、输出内容

系统输出三部分：

1.  改写后的完整笔记\
2.  小红书推荐正文（100--150 字）\
3.  标签

标签格式：

#标签 #标签 #标签
